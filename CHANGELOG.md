# Changelog

All notable changes to FontBlitz will be documented in this file.

## [1.0.0] - 2026-02-17

### Added
- Initial release of FontBlitz
- Deep-scan technology for nested ZIP files (up to 5 levels)
- Smart detection of font files (.ttf, .otf) inside archives
- Batch installation to Windows system fonts directory
- Automatic registry registration for installed fonts
- Duplicate detection and skipping
- Auto-elevation for administrator privileges
- Comprehensive logging system
- Zero data collection - 100% local operation
- Support for Windows 10 and 11
- Detailed installation summary with statistics

### Features
- Extracts only ZIPs containing fonts
- Leaves non-font archives untouched
- Handles corrupt and password-protected archives gracefully
- Full PowerShell 5.1+ compatibility
