# FontBlitz Repository Structure

```
FontBlitz/
│
├── .github/
│   └── FUNDING.yml              # GitHub Sponsors button config
│
├── .gitignore                   # Files to exclude from git
├── CHANGELOG.md                 # Version history and updates
├── CONTRIBUTING.md              # Guidelines for contributors
├── FontBlitz.ps1               # Main PowerShell script ⭐
├── GITHUB_SETUP.md             # Instructions for uploading to GitHub
├── LICENSE                      # MIT License
├── README.md                    # Main documentation
└── REPOSITORY_STRUCTURE.md     # This file
```

## File Descriptions

### Core Files
- **FontBlitz.ps1** - The main PowerShell script that does all the font installation magic
- **README.md** - Primary documentation shown on GitHub repository page

### Documentation
- **CHANGELOG.md** - Tracks all version changes and updates
- **CONTRIBUTING.md** - How to contribute to the project
- **GITHUB_SETUP.md** - Step-by-step guide to upload this to GitHub
- **REPOSITORY_STRUCTURE.md** - Overview of repository organization

### Configuration
- **LICENSE** - MIT License (free to use, modify, distribute)
- **.gitignore** - Tells git to ignore log files, fonts, and temp files
- **.github/FUNDING.yml** - Adds "Sponsor" button linking to Patreon

## What Gets Uploaded to GitHub

Everything in this folder should be uploaded to GitHub:
- ✅ All markdown (.md) files
- ✅ FontBlitz.ps1 script
- ✅ LICENSE file
- ✅ .gitignore file
- ✅ .github folder

What does NOT get uploaded (automatically ignored):
- ❌ Font files (.ttf, .otf, etc.)
- ❌ ZIP archives
- ❌ Log files
- ❌ Temporary files

See **GITHUB_SETUP.md** for detailed upload instructions.
