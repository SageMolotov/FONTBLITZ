# Contributing to FontBlitz

Thanks for your interest in improving FontBlitz! This project was built by Sage Molotov + Claude AI as a free tool for the creative community.

## How to Contribute

### Reporting Bugs
If you find a bug, please open an issue with:
- A clear description of the problem
- Steps to reproduce the issue
- Your Windows version and PowerShell version
- Any error messages from the log file

### Suggesting Features
Have an idea? Open an issue describing:
- What problem it solves
- How it would work
- Why it would be useful to the community

### Pull Requests
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test thoroughly on Windows 10 and/or 11
5. Commit with clear, descriptive messages
6. Push to your fork
7. Open a Pull Request with a detailed description

### Code Standards
- Keep the script readable and well-commented
- Maintain backward compatibility with PowerShell 5.1+
- Test on both Windows 10 and Windows 11
- Follow the existing code style
- Update documentation if needed

### Testing
Before submitting:
- Test with various ZIP structures (single, nested, mixed content)
- Verify duplicate detection works
- Check error handling for corrupt ZIPs
- Ensure logging captures all operations
